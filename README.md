# BigQuery SQL Practice

A collection of SQL queries written in Google BigQuery,
covering a range of topics from basics to real-world analysis.

## Topics Covered
- Filtering, sorting, aggregations
- JOINs (INNER, LEFT, RIGHT, FULL)
- Window functions 
- CTEs and subqueries
- Date functions
- Real-world analysis projects

## Tools Used
- Google BigQuery
- Google Cloud Console
